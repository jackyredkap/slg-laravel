## 🚀 Installation

Clone the project

```bash
git clone https://github.com/jackyredkap/slg-laravel.git
```

Go to project

```bash
cd slg-laravel
```

Install dependencies

```bash
composer install
```

Copy environment file

```bash
cp .env.example .env
```

Generate application key

```bash
php artisan key:generate
```

Run migration

```bash
php artisan migrate
```

Start server

```bash
php artisan serve
```
